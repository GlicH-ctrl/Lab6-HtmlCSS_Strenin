О SVG

SVG (Scalable Vector Graphics) — масштабируемая векторная графика, которая отображается без потери качества при любом увеличении.

Преимущества SVG:

  • Масштабируется без потери качества
  • Малый размер файла
  • Управление через CSS и JavaScript
  • Доступность (можно добавить title и description)

Атрибуты SVG:

  • width, height — размеры холста
  • fill — цвет заливки
  • stroke — цвет обводки
  • cx, cy — координаты центра круга
  • r — радиус круга
  • rx, ry — радиус скругления
